# Website design profile

The authoritative OpenE2EE design contract lives in
[`open-e2ee/design`](https://github.com/open-e2ee/design/blob/main/DESIGN.md).

- Shared release: `@open-e2ee/design` v0.1.0
- Surface: public marketing and engineering editorial
- Density: spacious and expressive
- Theme: `light`, `dark`, and `system` through the shared `.dark` root contract
- Local ownership: page composition, marketing components, editorial layouts,
  and product-specific copy

`src/styles/design-system.css` temporarily maps the website's established
unprefixed variables to shared semantic tokens. New components should use
`--oe-*` variables directly. Remove the compatibility aliases after the
existing component stylesheet has migrated.

Brand assets under `public/brand` are generated by `oe-design`; do not edit
them directly.
